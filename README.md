# 🌐 Personal Portfolio Website

A modern, responsive portfolio website built to showcase my work, technical skills, and experience as a Senior Software Engineer.

## ✨ Overview

This portfolio is designed with a strong focus on **clean UI, smooth interactions, and real-world usability**. It highlights selected projects, testimonials, and provides an easy way to connect with me.

The goal of this project is not just presentation — but to demonstrate **production-level front-end practices**, attention to detail, and thoughtful UX implementation.

---

## 🚀 Tech Stack

* **React (Vite)** – fast and modern frontend setup
* **CSS / Tailwind-style utility classes** – responsive and clean styling
* **Component-based architecture** – reusable and maintainable UI
* **Figma Make (AI-generated base)** – refined and customized into production-ready code

---

## 🎯 Key Features

* Responsive layout across desktop and mobile
* Interactive project showcase with modal previews
* Smooth hover animations and UI transitions
* Testimonial section with dynamic avatars
* Social links with accessible and clean UI
* Optimized static build for deployment

---

## 🧠 What I Focused On

* UI/UX consistency and visual hierarchy
* Clean component structure and reusability
* Performance and lightweight assets
* Real-world usability (not just design mock)

This portfolio represents how I approach frontend development:
